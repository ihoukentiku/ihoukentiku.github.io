## 加工内容

すべての SVG に共通して以下の加工を施しています:

1. 元の game-icons.net SVG は黒地白アイコン (背景 path + 白アイコン path) の構造
2. 背景 path (`<path d="M0 0h512v512H0z"/>`) を削除
3. アイコン本体の `fill="#fff"` を `fill="#505050"` に変更 (白背景での視認性のため)

ピッカーのフィル/ストロークで色を上書きできます。
