                    使い方ガイド
                    
                

                

                    

                        
基本的な流れ

                        

                            add_circle
                            

                                
NPCを追加する

                                
左パネルの「＋ 新規」ボタンで NPC を追加。リストの項目をクリックして切り替えます。不要な NPC は ✕ で削除できます（1体は残ります）。

                            

                            tune
                            

                                
バージョンを選ぶ

                                
EDIT パネル右上の CoC7 / CoC6 で版を切り替えます。切り替えると派生値（HP・DB など）が自動で再計算されます。

                            

                            casino
                            

                                
能力値を入力 / ロールする

                                

                                    ダイス欄に式（例：3D6）を入力して各
                                    casino ボタンで個別ロール、または「全ロール」で一括ロールできます。CoC7
                                    では結果が自動的に×5されます。
                                

                            

                            monitor_heart
                            

                                
派生値を確認・編集する

                                
HP・MP・DB などは能力値から自動計算されます。手動で上書きすることも可能です。SAN トグルで正気度の出力のオン／オフを切り替えられます。

                            

                            psychology
                            

                                
技能を追加する

                                
「技能を追加」ボタンで技能行を追加。技能名と成功値（%）を入力します。CoC6 の場合は判定タイプ（CC / CCB）も選択してください。

                            

                            code
                            

                                
コマンド / ダメージを追加する

                                

                                    武器攻撃などのダイス式を登録します。式中に DB・db
                                    と書くとダメージボーナスの参照に変換されます（例：1D6+2+DB）。
                                

                            

                        


                        
出力パネル

                        

                            data_object
                            

                                
JSON タブ

                                
ココフォリアなどにそのまま貼り付けられるキャラクター JSON を出力します。「コピー」ボタンでクリップボードにコピーできます。

                            

                            chat
                            

                                
チャパレ タブ

                                
BCDice 用チャットパレットを出力します。能力値・技能・コマンド・ステータスコメントが含まれます。

                            

                            replay
                            

                                
ロール＆コピー

                                
全能力値を一括ロールしてから即座にコピーします。NPCを量産する際に便利です。

                            

                        


                        
データの保存

                        

                            save
                            

                                
自動保存

                                
入力内容はブラウザのローカルストレージに自動保存されます。ページを閉じても次回アクセス時に復元されます。

                            

                            warning
                            

                                
注意

                                
ブラウザのデータを削除するとNPCデータも消えます。重要なキャラクターは JSON をコピーして手元に保管してください。
